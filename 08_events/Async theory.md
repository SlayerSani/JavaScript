# Javascript: 
### Default

```
1. Synchronous
2. single thread
```

# Execution Context
```
Executes one line of code at a time
example:
console.log(1);
console.log(2);

---> Each operation waits for the last one to complete before executing
(a)Call Stack
(b)Memory Heap
```

# Blocking code vs Non Blocking code
```
Blocking Code:
1. Block the flow of the program
2. Read File Sync


Non Blocking Code:
1. Does not Block execution
2. Read file Async
```
